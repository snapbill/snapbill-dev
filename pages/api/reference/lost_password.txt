Lost Password
=============

Recover your account's password using either the username or email address.

<section class="api-call" id="username" data-title="/username" markdown=1>
### /v1/lost_password/username

Recover the password of the account associated with the given username. Returns a list of matched usernames.

</section>


<section class="api-call" id="email" data-title="/email" markdown=1>
### /v1/lost_password/email_address

Recover the password of the account associated with the given email address. Returns a list of matched email addresses.

</section>

<section class="parameters" id="parameters" data-title="parameters">
### Parameter listing ###

email: email
	The email address of the account to recover.

username: username
	The username of the account to recover.

</section>
