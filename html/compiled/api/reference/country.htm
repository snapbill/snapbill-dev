Countries

/v1/country/code/get

Get the specified country. 'code' must be the iso2 code.

Parameter listing

NameAvailabilityDescription
depthget The depth parameter signals the amount of data for the country that was provided. depth=0 is the highest level that provides full detail.
codeget The unique ISO 2 code two letter code for the country, which is always specified in upper case.
iso2get See 'code'.
iso3get The unique ISO 3 code three letter code for the country, which is always specified in upper case.
nameget The name of the country.

/v1/country/list

List all countries.

Results can optionally be filtered by the filters listed below.

List Countries Filters

NameAvailabilityDescription
iso2get The iso2 code (see above) of the counties.
pageget The page of results to return.
perpageget The number of results per page to return.
