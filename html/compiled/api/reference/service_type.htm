Parameter listing

NameAvailabilityDescription
depthget The depth parameter signals the amount of data for the payment method that was provided. depth=0 is the highest level that provides full detail.
idget The id of this service type. Unique over a single accounts.
xidget The xid of the service type. This is guaranteed to be unique over all accounts.
nameget The name of the service type as chosen by the user.
