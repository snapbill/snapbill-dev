Add Service Parameters

NameAvailabilityDescription
service_type_idadd The id of the service to add. This field is required
packageadd The name of the package to use in the service. This field is required
termadd The renewal term. This field is required
data-*customfieldname*add Optionally specify custom fields for a package.

Parameter listing

NameAvailabilityDescription
depthget The depth parameter signals the amount of data for the service that was provided. depth=0 is the highest level that provides full detail.
idget The id of the service in the system. This is guaranteed to be unique only over a single account.
xidget The xid of the service in the system. This is guaranteed to be unique over all accounts (see Concepts - Xid)
