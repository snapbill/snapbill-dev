Job Offer

SnapBill does not offer a programmatic way to add job offers, if you would like this feature, please contact us.

/v1/job_offer/list

List all job offers in the account. Valid filters are page and perpage.

List Job Offers Filters

page: get The page of results to return.

perpage: get The number of results to return per page.

Parameter list

NameAvailabilityDescription
titleget The title of the position on offer.
descriptionget A description of the position on offer.
locationget The location of the position on offer.
remote_optionget A field representing whether a given offer will accept remote candidates.

/v1/job_offer/id/apply

Apply for a job by id. There is a verification to prevent abuse. This route uses the id of the job_offer as opposed to the xid.

Parameters for Job Application

NameAvailabilityDescription
name The name of the applicant.
email The email address of the applicant.
verify The verification string.
